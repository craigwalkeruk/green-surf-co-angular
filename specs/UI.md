# Design

The following Figma file is the source of truth for UI design

https://www.figma.com/design/VzZx5Jpt65qkq1jlau2oFv/POC-Build-from-scratch-in-WebStorm?node-id=324-11135&m=dev

Figma MCP is installed and can be used to get design when generating and testing codes correctness.

Storybook and storybook MCP should be used to create and test all UI components so they follow the Figma design exactly

All AI agents should use Jetbrains MCP to access the terminal do only use command line actions via the JetBrains IDE terminal is the JetBrains MCP tool for your use case is not present. See: https://www.jetbrains.com/help/webstorm/mcp-server.html#supported-tools
